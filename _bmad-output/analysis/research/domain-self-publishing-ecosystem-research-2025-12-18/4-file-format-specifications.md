# 4. File Format Specifications

## 4.1 EPUB Standard (Industry Requirement)

**Why EPUB Matters:**
- **Amazon shifted to EPUB** as primary format in 2022 ([LivingWriter](https://livingwriter.com/blog/how-to-publish-a-book-on-amazon-2025-kdp-guide/))
- "EPUB files offer **superior flexibility** for reflowable content" ([HMD Publishing](https://hmdpublishing.com/amazon-kdp-formatting/))
- Text adapts seamlessly to different screen sizes and font preferences
- **EPUB3** is current standard (multimedia support, accessibility features)

**EPUB vs. MOBI:**
- MOBI is **discontinued** (Amazon dropped support March 2025)
- EPUB is **universal standard** (works across all platforms)
- EPUB supports better formatting, embedded fonts, advanced styling

## 4.2 PDF Requirements for Print

**Print PDF Specifications:** ([KDP Help](https://kdp.amazon.com/en_US/help/topic/G201857950))
- **Resolution**: 300 DPI minimum (all content)
- **Color space**: CMYK for color images, Grayscale for B&W
- **Fonts**: All fonts embedded
- **Transparencies**: Flattened (prevents print errors)
- **Bleed**: 0.125" (3.175mm) on all sides if using bleed
- **PDF/X-1a**: Preferred standard for print-ready files

**Common Print Sizes:**
- **5" x 8"**: Standard novel size
- **6" x 9"**: Trade paperback, nonfiction
- **8.5" x 11"**: Textbooks, workbooks
- **Custom sizes**: Supported within KDP ranges

## 4.3 Metadata Requirements

**Essential Metadata Fields:** ([Manuscript Report](https://manuscriptreport.com/blog/7-proven-steps-to-optimize-book-metadata-for-amazon), [Author Imprints](https://www.authorimprints.com/book-metadata-optimization-primer-self-publishing-authors-businesses/))

1. **Title**: Exact match, front-loaded keywords
2. **Subtitle**: Additional keywords, clear value proposition
3. **Author Name**: Consistent across all titles (brand building)
4. **Series Information**: Series name, book number
5. **Description**: HTML formatted, 4000 characters max
6. **Publisher**: Author name or imprint
7. **Publication Date**: Controls "new release" status
8. **Language**: Primary language of text
9. **ISBN**: Optional for KDP, required for wide distribution
10. **Categories**: Up to 2 on KDP (can request up to 10)
11. **Keywords**: 7 keyword boxes, 50 characters each
12. **Age Range**: For children's books
13. **BISAC Codes**: Standard book industry codes
